# 火影忍者主题网页设计项目

本项目是一个以《火影忍者》为主题的静态网页设计作品，涵盖了首页、角色介绍、精彩剧情、图片鉴赏以及用户注册功能。

## 项目预览

- **首页 (index.html)**: 展示火影忍者的基本信息、情景介绍和核心剧情。
- **角色介绍 (gzp1.html)**: 详细列举了火影忍者中的主要角色及其背景。
- **精彩剧情 (gzp2.html)**: 回顾作品中的经典战斗和感人瞬间。
- **图片鉴赏 (gzp3.html)**: 收集了高质量的火影相关图片。
- **用户注册 (gzp4.html)**: 完善的忍者登记表单，支持选择所属村落和性别。
- **注册成功 (cg.html)**: 精美的忍者登记成功反馈页面。

## 核心优化记录

### 1. 移动端适配优化
- 为所有 HTML 页面添加了 `<meta name="viewport" content="width=device-width, initial-scale=1.0">`，解决了 Microsoft Edge Tools 的警告，确保页面在不同设备上的缩放正常。

### 2. 注册功能完善
- **表单升级**: 在 [gzp4.html](page/gzp4.html) 中增加了“性别”单选框和“所属村落”下拉选择框（包括木叶村、砂隐村等）。
- **交互提升**: 添加了注册成功后的弹窗提示。

### 3. 视觉体验升级
- **成功页面美化**: 重新设计了 [cg.html](page/cg.html)，引入了卡片式布局、主题背景图以及具有火影特色的金色配色方案，增强了用户的沉浸感。

## 技术栈
- **HTML5**: 结构化内容组织。
- **CSS3**: 负责整体布局、阴影效果、圆角及响应式基础。
- **JavaScript**: 简单的交互逻辑（如注册成功弹窗）。
